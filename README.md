🚀 Getting Started
1️⃣ Clone the Repository
git clone https://github.com/MehirLallieEmeris/Ice-Task-3-and-4.git
2️⃣ Open in Android Studio
Use Android Studio Ladybug or newer
Click: File > Open → Select project folder
3️⃣ Sync Project
Wait for Gradle sync
Click "Sync Project" if prompted
4️⃣ Run the App
Start emulator OR connect device
Click ▶️ Run
⚠️ Known Limitations

⚡ Local Data Only
Uses a static gems.json file (no live API)

📉 Static Pricing
Estimates may not reflect real-world price changes

🗺️ No Maps Integration
Locations are listed but not linked to Google Maps

👤 No User Accounts
Saved trips are stored locally (not synced)
